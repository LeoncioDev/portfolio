## Features

- Cursor glow that follows the mouse — disabled on touch devices
- Scroll spy — nav highlights the active section automatically
- Reveal animations — elements fade and slide up as they enter the viewport
- Header shadow on scroll
- Fully responsive — mobile breakpoints for all sections
- No JavaScript frameworks, no build step — open index.html and it works

## Design

- **Background:** `#080c10` — near-black with a subtle blue tint
- **Accent:** `#22d3ee` — electric cyan, applied to highlights, links and interactive elements
- **Fonts:** Sora (text) + JetBrains Mono (UI labels, code, nav)
- **Noise texture:** SVG inline `feTurbulence` filter for depth without external images
- **Sections:** numbered 01–04 with a decorative line extending to the right
